- Running backend
```
clone repo
```
- cd in repo
```
cd reporting-backend
```
- copy .env.exampe to .env 
```
cp .env.example .env
```
-install package
```
composer install
```
- generate key
```
php artisan key:generate
```
- Make migration
```
php artisan make:migration
```
- run server
```
php artisan serve
```
